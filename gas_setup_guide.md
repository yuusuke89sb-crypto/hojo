# 📋 Google Apps Script セットアップガイド

ヒアリングシートのデータをGoogleスプレッドシートに送信する機能のセットアップ手順です。

---

## ステップ 1: Google スプレッドシートを作成

1. [Google スプレッドシート](https://sheets.google.com) を開く
2. 「空白」で新しいスプレッドシートを作成
3. タイトルを「**補助者ヒアリングシート回答**」に変更
4. **Sheet1 の1行目（ヘッダー行）に以下をコピペ**：

```
送信日時	氏名	フリガナ	生年月日	性別	住所	電話番号	メールアドレス	緊急連絡先	前職事務所名	勤務年数	前職雇用形態	担当業務内容	車庫証明経験	車関係スキル	訪問経験	PCスキル	運転免許	自家用車	行政書士資格	その他資格	希望勤務日数	希望勤務時間帯	希望時給	勤務開始可能日	通勤手段	健康状態	引継ぎ事項	その他特記事項
```

> ⚠️ タブ区切りです。A1〜AC1に自動的に分かれます。

---

## ステップ 2: Google Apps Script を作成

1. スプレッドシートのメニューから **「拡張機能」→「Apps Script」** をクリック
2. エディタが開いたら、既存のコードを **すべて削除**
3. 下記の `google_apps_script.js` の内容を **すべてコピーして貼り付け**
4. **プロジェクト名** を「補助者ヒアリング受付」に変更

---

## ステップ 3: ウェブアプリとしてデプロイ

1. Apps Script エディタで **「デプロイ」→「新しいデプロイ」** をクリック
2. 左の歯車アイコンをクリックして **「ウェブアプリ」** を選択
3. 設定を以下のようにする：
   - **説明**: 「ヒアリングシート受付」
   - **次のユーザーとして実行**: 「自分」
   - **アクセスできるユーザー**: 「**全員**」
4. **「デプロイ」** をクリック
5. 「アクセスを承認」をクリック → 自分のGoogleアカウントを選択
6. 「詳細」→「（安全ではないページ）に移動」→「許可」をクリック
7. 表示された **ウェブアプリURL** をコピー

> 📌 URLは `https://script.google.com/macros/s/xxxxx/exec` の形式です

---

## ステップ 4: URLを設定

1. `script.js` ファイルを開く
2. **3行目** の以下の部分を見つける：
   ```javascript
   const GAS_URL = '';
   ```
3. コピーしたURLを貼り付ける：
   ```javascript
   const GAS_URL = 'https://script.google.com/macros/s/あなたのID/exec';
   ```
4. ファイルを保存

---

## ✅ 完了！

これで、ヒアリングシートの「📤 送信」ボタンを押すと：
1. 入力データがGoogleスプレッドシートに自動保存されます
2. 送信成功のメッセージが表示されます
3. スプレッドシートで全ての回答を一覧できます

---

## 🔧 トラブルシューティング

| 症状 | 対処法 |
|---|---|
| 送信エラーが出る | Apps ScriptのURLが正しいか確認 |
| 権限エラー | 「アクセスできるユーザー」が「全員」になっているか確認 |
| データが届かない | Apps Scriptのエディタで「実行ログ」を確認 |
| URLを変更したい | 「デプロイ」→「デプロイを管理」からURLを確認 |

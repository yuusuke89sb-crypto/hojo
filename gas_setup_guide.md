# 📋 Google Apps Script セットアップガイド

ヒアリングシートのデータをGoogleスプレッドシートに送信し、  
**履歴書・雇用契約書を自動生成するリンク**も作成されます。

---

## ステップ 1: Google スプレッドシートを作成

1. [Google スプレッドシート](https://sheets.google.com) を開く
2. 「空白」で新しいスプレッドシートを作成
3. タイトルを「**補助者ヒアリングシート回答**」に変更
4. **A1セル**に以下をコピペ（タブ区切りで自動的に列に分かれます）：

```
送信日時	氏名	フリガナ	生年月日	性別	住所	電話番号	メールアドレス	緊急連絡先	前職事務所名	勤務年数	前職雇用形態	担当業務内容	車庫証明経験	車関係スキル	訪問経験	PCスキル	運転免許	自家用車	行政書士資格	その他資格	希望勤務日数	希望勤務時間帯	希望時給	勤務開始可能日	通勤手段	健康状態	引継ぎ事項	その他特記事項	📄履歴書リンク	📝雇用契約書リンク
```

---

## ステップ 2: Google Apps Script を作成

1. スプレッドシートのメニューから **「拡張機能」→「Apps Script」** をクリック
2. エディタが開いたら、既存のコードを **すべて削除**
3. `google_apps_script.js` の内容を **すべてコピーして貼り付け**
4. **⚠️ 重要：3行目の `SITE_URL` をあなたのGitHub PagesのURLに変更**：
   ```javascript
   var SITE_URL = 'https://あなたのユーザー名.github.io/補助者名簿';
   ```
5. **プロジェクト名** を「補助者ヒアリング受付」に変更

---

## ステップ 3: ウェブアプリとしてデプロイ

1. Apps Script エディタで **「デプロイ」→「新しいデプロイ」** をクリック
2. 左の歯車アイコンをクリックして **「ウェブアプリ」** を選択
3. 設定：
   - **説明**: 「ヒアリングシート受付」
   - **次のユーザーとして実行**: 「自分」
   - **アクセスできるユーザー**: 「**全員**」
4. **「デプロイ」** をクリック
5. 「アクセスを承認」→ 自分のGoogleアカウント → 「詳細」→「移動」→「許可」
6. 表示された **ウェブアプリURL** をコピー

---

## ステップ 4: URLを設定

1. `script.js` を開く
2. **5行目** の `GAS_URL` にコピーしたURLを設定：
   ```javascript
   const GAS_URL = 'https://script.google.com/macros/s/あなたのID/exec';
   ```
3. 保存 → Git commit & push

---

## ✅ 使い方

### ヒアリングシート送信
補助者候補がフォームに入力 → 「📤 送信」→ スプレッドシートに保存

### 書類作成（スプレッドシートから）
スプレッドシートの右端の列にリンクが自動生成されます：
- **�履歴書リンク** → クリックで履歴書が自動反映
- **📝雇用契約書リンク** → クリックで雇用契約書が自動反映

そのまま印刷/PDF保存で完成！
